# leetcode-cpp

# 一、整体策略（非常重要）
C++ 软件开发岗面试，算法题特点：
- 重点考：数组、链表、栈队列、二叉树、哈希、二分、DFS/BFS、动态规划基础
- 高频附带：代码规范、边界处理、STL 使用、复杂度分析

刷题顺序建议：
基础数据结构 → 搜索与回溯 → 贪心/二分 → 简单 DP → 高频经典题
