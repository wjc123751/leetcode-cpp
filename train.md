## DFS
1. 标准DFS栈模板 弹出再处理
起点入栈
标记起点已访问
while(栈非空){
    u = 栈弹出      // 弹出再处理
    访问u
    for( 邻接节点 v 逆序 ){
        if(v 未访问){
            标记已访问
            v 入栈
        }
    }
}
2. 回溯DFS栈模板 先留栈，处理两遍，最后弹出
栈.push(起点);
while(栈不为空)
{
    node = 栈.top(); // 关键点：只取栈顶，不弹出

    if(当前节点未访问){
        标记为已访问;
        逆序/顺序压入合法邻居;
    }
    else{
        恢复原状;
        栈.pop(); // 第二次才弹出
    }
}
